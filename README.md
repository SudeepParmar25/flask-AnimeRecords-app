# Anime Tracker Web Application

A Flask-based web application that allows users to search, track, and manage anime records.

## Features
- User authentication
- Anime record management
- Search functionality
- MySQL database integration
- Responsive UI

## Tech Stack
- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript
